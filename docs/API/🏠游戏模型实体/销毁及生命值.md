
### 属性

#### entity.destroyed : boolean  
默认值：false

**如果为真(true)，实体就被销毁。**

---


#### entity.enableDamage : boolean  
默认值：false

**如果为真true，则可对实体进行伤害。**

---


#### entity.**showHealthBar** **: **boolean  
默认值：true

**如果为真true，则显示实体的生命值HP。**

---


#### entity.hp **: **number 
默认值：100

**实体的当前生命值hp。**

---


#### entity.maxHp **: **number 
默认值：100

**实体的最大生命值hp。**
